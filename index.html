Game-2

This is my own game based on a 2048 game plot. In this game, the player must combine tiles containing the same numbers until they reach the number 2560. The tiles can contain only integer values starting from 2,3,5 and those are multiples of 5, like 5, 10, 20,40 , 80, and so on. Ideally, the player should reach the 2560 tile within the smallest number of steps. In every step when you combine the similar tiles, the scores will be increased. The board has dimension of 4 x 4 tiles, so that it can fit up to 16 tiles. If the board is full, and there is no possible move to make like merging tiles together - the game is over.
